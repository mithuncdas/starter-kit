Conventions

Response envelope

Every JSON response follows the same shape.

{
  "success": true,
  "message": "Human readable message.",
  "data": { ... }   // object, array, or paginator
}

Auth header

Send the Sanctum token on every protected route.

Accept: application/json
Authorization: Bearer 1|xxxxxxxx...

Token comes from POST /admin/login → data.token.

Status codes

  • 200 OK
  • 201 Created
  • 401 Unauthenticated (missing/invalid token)
  • 403 Forbidden (missing permission / self-protection / deactivated)
  • 404 Not found
  • 422 Validation error
  • 429 Too many requests (rate limited)

Validation error (422)

{
  "success": false,
  "message": "The email field is required.",
  "errors": {
    "email": ["The email field is required."]
  }
}
